# 🎉 YTGrab v2.1.0 - First Release

**YTGrab**（旧称: YouTube Downloader）は、YouTubeから動画や音声をダウンロードするための強力で使いやすいツールです。このリリースでは、UXの大幅な改善と新機能の追加が行われました。

## ✨ 新機能

- **📚 プレイリスト選択機能**: ダウンロード前にプレイリストから特定の動画を選択できるようになりました。プレイリスト全体をダウンロードする必要はありません！
- **⏳ ローディングオーバーレイ**: 動画情報の取得など、時間のかかる処理中に視覚的なローディング表示を追加しました。誤操作を防ぎ、処理状況がわかりやすくなりました。
- **🎨 ブランド変更**: アプリケーション名が正式に **YTGrab** になりました。
- **🚀 スタンドアロン実行**: Pythonをインストールせずに動作します。依存関係（yt-dlp, ffmpeg）は初回起動時に自動的にダウンロードされます。

## 🛠️ 主な機能

- **動画**: 4K, 1080p, 720pなどでダウンロード可能 (MP4, WebM, MKV)
- **音声**: 高品質な音声抽出 (MP3, M4A, Opus)
- **字幕**: 自動生成または手動作成された字幕のダウンロード
- **サムネイル**: 動画への埋め込みや保存が可能
- **高度な機能**: プロキシサポート、Cookie、ファイル名テンプレートなど

## 📦 インストール方法

1. 下の **Assets** セクションから **`YouTubeDownloader.exe`** をダウンロードしてください。
2. ダウンロードした実行ファイルを起動します。
3. *注意*: 初回起動時は、必要なコンポーネント（yt-dlp/ffmpeg）のダウンロードに数分かかる場合があります。
